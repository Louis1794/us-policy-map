## App Description
Welcome!! Before you sits the amazing USA Policy and news Map! We bring an interactive US map where users click any state to view the latest  policy news headlines for that state, submit their own policy opinions, and read opinions submitted by others — all stored and retrieved from MongoDB. Any questions just let us know in our contact emails below!!
